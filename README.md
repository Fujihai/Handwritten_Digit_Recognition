# Handwritten_Digit_Recognition
基于OpenCV的手写数字识别 

基本操作：通过鼠标在窗体上面书写数字，程序能识别当前的数字。  

#-------------------------------------------------------------------

	详细操作说明(按键)：

	缩小笔迹宽度：按‘-’键
	增加笔迹宽度：按‘+’键
	清除画板	   ： R键
	退出程序	   ： Esc键
	保存当前绘制的数字图像： s键(默认保存在E盘(E:/out.pbm))
  
#--------------------------------------------------------------------

运行环境：

	Windows 10操作系统
  
	QT Creator 4.2.1(Community)
  
	OpenCV 2.4.10
