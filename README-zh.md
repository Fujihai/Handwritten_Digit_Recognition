# 手写数字识别

介绍
---------------------

基于OpenCV的手写数字识别程序(主要使用k最邻近分类算法)。

在窗体上使用鼠标书写数字，程序能识别出来你所书写的数字。

运行环境
------------------------

	Windows10 OS

	QT Creator 4.2.1(Community)

	OpenCV 2.4.10

使用说明
-----------------------
-------------------------------------------------------------------

	详细操作说明(按键)：

	缩小笔迹宽度：按‘-’键
	增加笔迹宽度：按‘+’键
	清除画板	   ： R键
	退出程序	   ： Esc键
	保存当前绘制的数字图像： s键(默认保存在E盘(E:/out.pbm))
